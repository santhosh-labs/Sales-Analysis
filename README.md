# 📊 Sales Performance Dashboard – Super Market

## 📌 Project Overview

This project is an interactive **Microsoft Excel Sales Performance Dashboard** built using a Super Market sales dataset. The dashboard provides a comprehensive view of sales performance across regions, states, customer segments, categories, shipping modes, cities, and product sub-categories.

The objective of this dashboard is to help business users identify high-performing products, profitable regions, customer purchasing patterns, and shipping preferences for better decision-making.

---

# 🎯 Objectives

- Analyze overall sales performance.
- Compare sales across different regions and states.
- Identify top-performing cities.
- Evaluate sales by customer segment.
- Analyze category and sub-category performance.
- Compare shipping mode contributions.
- Provide an interactive dashboard using Excel slicers.
